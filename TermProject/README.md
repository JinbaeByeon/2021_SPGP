# 2015182013 변진배
## **2021-1 SPGP TermProject** - 스마트폰 게임 프로그래밍 텀 프로젝트

### 게임 컨셉
- 아래에서 위로 진행하는 종 스크롤 방식의 탄막 슈팅게임
- 플레이어는 위에서 나오는 적 객체들을 파괴하여 점수를 얻음
- 적들의 총알에 맞아 체력이 0이 되면 게임 오버
 
---------------------------------------------------------------------------------------------------------------

### 개발 범위
- 2D 애니메이션

- 배경 스크롤

- 터치이벤트를 통한 플레이어 이동 및 공격

- 총알 - 플레이어 - 적기 사이의 충돌처리

- 사운드 처리
